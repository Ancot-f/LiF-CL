## 二、背景资料

### 资料 A：机器学习理论基础（来源：李凡长《机器学习理论及应用》）

本书建立了机器学习理论的完整框架，你需要从中提取以下关键数学概念作为 PPT 前 1/3 的理论基础：

**概念1：PAC 学习理论（Probably Approximately Correct）**

- 核心思想：一个概念类是 PAC 可学习的，当且仅当存在一个算法，对任意 ε, δ > 0，当训练样本数 m 足够大时，学习器能以至少 1-δ 的概率输出一个泛化误差不超过 ε 的假设
- 关键公式：样本复杂度下界 $m \geq \frac{1}{\epsilon}(\ln|H| + \ln\frac{1}{\delta})$
- 需要强调的洞察：PAC 理论保证泛化的前提是——训练集和测试集来自**同一个分布 D**。这是所有后续讨论的逻辑起点。

**概念2：VC 维（Vapnik-Chervonenkis Dimension）**

- 核心思想：度量假设空间的"表达能力"——一个假设空间 H 的 VC 维是能被 H 打散的最大样本数
- 关键公式：$m \geq \frac{1}{\epsilon}(d(\ln\frac{2m}{d}+1) + \ln\frac{4}{\delta})$，其中 d 是 VC 维
- 需要强调的洞察：VC 维越大 → 模型表达能力越强 → 但泛化所需的样本也越多。这为后续讨论"模型按需增容"提供了理论伏笔。

**概念3：结构风险最小化（Structural Risk Minimization, SRM）**

- 核心思想：将假设空间按 VC 维组织成嵌套结构 S₁ ⊂ S₂ ⊂ ... ⊂ Sₖ，在经验风险和置信范围之间寻找最优平衡
- 关键公式：$R(f) \leq R_{emp}(f) + \Phi(\frac{h}{n})$
- 需要强调的洞察：当数据量固定时，不能无限制增大模型容量。在持续学习中，"何时增容"成为需要严格理论依据的决策。

**概念4：独立同分布假设（i.i.d. Assumption）的统治地位**

- PAC、VC维、SRM 的全部理论保证都建立在训练数据和测试数据独立同分布这一假设之上
- 这是经典统计学习理论的"阿基米德支点"——抽掉它，整个框架需要被重新审视
- 也是我们要系统性地挑战和扩展的假设

### 资料 B：李群与纤维丛的数学基础

以下是从李凡长《机器学习理论及应用》中李群纤维丛学习部分提取的核心数学结构。你需要将这些内容融入 PPT 的理论部分。

**概念5：李群（Lie Group）基础**

- 定义：李群 G 是一个既是群又是光滑流形的数学对象，群运算（乘法和取逆）是光滑映射
- 对机器学习的关键意义：李群同时具有代数结构（群运算 = 参数组合）和几何结构（流形 = 参数空间的局部欧氏性质），这使它成为描述"结构化参数空间"的自然语言
- 矩阵李群的核心例子：
  - 一般线性群 GL(n)：所有 n×n 可逆矩阵
  - 特殊正交群 SO(n)：{R ∈ ℝ^{n×n} | R^T R = I, det(R) = 1}
  - Stiefel 流形 St(d,r)：{W ∈ ℝ^{d×r} | W^T W = I_r}
- Stiefel 流形的齐性空间表示：St(d,r) ≅ SO(d) / SO(d-r)，即它是 SO(d) 的一个商空间。这个表示意味着 Stiefel 流形继承了一个自然的黎曼度量——由 SO(d) 的 Killing 形式诱导

**概念6：Stiefel 流形的黎曼几何**

- 切空间：在 W ∈ St(d,r) 处的切空间为 T_W St(d,r) = {ξ ∈ ℝ^{d×r} | ξ^T W + W^T ξ = 0}，即所有使正交约束一阶保持的扰动方向
- 黎曼度量：⟨ξ, η⟩_W = tr(ξ^T (I - ½W W^T) η)，它是 SO(d) 上 Killing 形式在商空间上的诱导度量
- 测地线方程：Stiefel 流形上存在闭合形式的测地线（由 Edelman et al., 1998 给出），这允许我们精确计算流形上两点之间的最短路径
- 测地线距离公式：
  - 设 W₁, W₂ ∈ St(d,r)，计算 W₁^T W₂ ∈ ℝ^{r×r} 的奇异值 σ₁, ..., σᵣ
  - 主角度（principal angles）：θᵢ = arccos(σᵢ)，其中 σᵢ = cos(θᵢ)
  - 测地线距离：$d_{geo}(W_1, W_2) = \sqrt{\sum_{i=1}^{r} \theta_i^2}$ （主角度向量的 ℓ² 范数）
- 几何直觉：两个 Stiefel 点之间的测地线距离度量了它们所张成的 r 维子空间之间的"夹角"。θᵢ = 0 表示两个子空间重合；θᵢ = π/2 表示正交。测地线距离是这些"子空间差异"的综合度量

**概念7：纤维丛（Fiber Bundle）理论框架**

- 定义：纤维丛 E 是一个五元组 (E, B, π, F, G)，其中：
  - E：全空间（Total Space）— 所有的模型参数
  - B：底流形（Base Space / Base Manifold）— 任务分布空间，每个点代表一种数据分布
  - π: E → B：投影映射 — 将参数映射到其适用的任务
  - F：标准纤维（Typical Fiber）— 在单个任务上的模型参数空间
  - G：结构群（Structure Group）— 一个李群，作用于 F 上，描述不同纤维之间的变换关系
- 局部平凡化（Local Trivialization）：对 B 的每个局部邻域 U，π^{-1}(U) ≅ U × F。即局部来看，纤维丛就是笛卡尔积——这意味着在相似的分布上，参数空间具有相同的结构
- 截面（Section）：映射 s: B → E，满足 π ∘ s = id_B。在机器学习中，一个"学习算法"就是在底流形（任务分布空间）上构造一个截面——为每个任务分配一组参数
- 联络（Connection）：定义了相邻纤维之间的"平行"关系，允许我们在底流形上沿路径"平行移动"纤维中的元素

**概念8：纤维丛视角下的持续学习**

- 底流形 B = 任务分布空间：每个任务对应底流形上的一个点。相似的任务在底流形上彼此靠近，差异大的任务远离
- 纤维 F_p = 在任务 p 上的最优参数：对于每个任务分布 p ∈ B，存在一组最优的 Adapter 参数。这组参数位于 p 上方的纤维中
- 结构群 G = SO(d)（或其在 Stiefel 流形上的作用）：描述不同任务参数之间的合法变换。G 的结构决定了参数空间的几何——测地线距离、平行移动、曲率
- 持续学习的纤维丛表述：
  - 顺序学习任务 T₁, T₂, ..., Tₜ 等价于在底流形上走一条分段路径 p₁ → p₂ → ... → pₜ
  - 在每个新任务 pₜ₊₁ 处，需要构造一个截面 s(pₜ₊₁)（即找到该任务的参数）
  - 核心问题：s(pₜ₊₁) 应该在多大程度上复用已有截面的信息（即旧 Adapter），还是应该在纤维中"离开"已有的截面（即触发扩展）？
  - 测地线距离回答了这个问题：如果新参数与所有旧参数在纤维上的测地线距离都很大（超过阈值），说明新任务在底流形上距离所有旧任务都远——需要新的截面分支（新 Adapter）
- 与经典理论的统一：
  - PAC 的 i.i.d. 假设：对应到底流形上，就是假设所有数据来自同一个点 p。在这种退化情况下，纤维丛退化为一个平凡的笛卡尔积，不需要联络，不需要平行移动
  - SEMA 的分布偏移检测（Z-score）：是用统计方法估计底流形上的两个点是否"足够近"
  - Lie-SEMA 的测地线距离检测：直接用纤维丛的内蕴几何来判断——不需要统计估计，距离由结构群 G 的黎曼度量直接给出

### 资料 C：SEMA 与 Lie-SEMA 的具体实现

**SEMA（Self-Expansion with Mixture of Adapters）— CVPR 2025**

三个核心组件：

1. **Functional Adapter**：瓶颈 MLP（768 → 16 → 768），插入 ViT 第 9-11 层的 MLP 旁
2. **Representation Descriptor（RD）**：轻量自编码器（768 → 128 → 768），用 Z-score 检测分布偏移
3. **Soft Router**：Linear(768 → M) → softmax，动态加权组合所有 Adapter 的输出

训练流程：检测（判断是否扩展）→ func 阶段（5 epoch, CE loss）→ rd 阶段（20 epoch, MSE loss）→ 冻结

**Lie-SEMA（李群改进版）**

核心改进：将 Adapter 的 down_proj 权重 W ∈ ℝ^{768×16} 约束到 Stiefel 流形 St(768,16) = {W | W^T W = I₁₆}

三个关键变化：

1. **参数初始化**：从 Haar 分布采样随机正交矩阵（QR 分解），而非 Kaiming Uniform
2. **优化约束**：每次 SGD 步后执行 SVD 投影 W ← U V^T，将 W 拉回 Stiefel 流形表面。这等价于 Riemannian SGD
3. **扩展检测升级**：从 Z-score（统计检测）→ 测地线距离（几何检测）
   - 计算公式：新 Adapter 训练完成后，计算 `min_i d_geo(W_new, W_i)`，若 > geo_threshold → 触发扩展
   - 本质上是在纤维中测量新旧"截面"的内蕴距离，而非依赖 AE 的间接统计

- 问题：欧氏距离 ‖W₁ - W₂‖_F 可能很小，但描述的特征子空间完全不同
- 引入约束：要求 W^T W = I_r → 参数不再生活在 ℝ^{d×r} 中
- Stiefel 流形定义：St(d,r) = {W ∈ ℝ^{d×r} | W^T W = I_r}
- 几何直觉：W 的 r 个列向量是 ℝ^d 中的正交归一基。W 指定了一个 r 维子空间（附带基底选择）。不同 Adapter 学习不同的 r 维"视角"来投影特征
- 测地线 vs 直线：欧氏距离的直线穿过流形内部（无意义），测地线沿流形表面（合法路径）

**建议图示**：左侧=欧氏空间中两个点之间的直线（红色虚线穿过流形内部，标注"‖W₁-W₂‖_F = 小，但子空间差异大"）。右侧=Stiefel 流形（弯月形曲面），上面两个点，测地线沿曲面连接（实线），标注"d_geo(W₁,W₂) = 大"。两个 W 用小方阵图标表示，标注维度 [768×16]。

**讲解词核心要点**（约 400 字）：当我们约束 W 到 Stiefel 流形，一个 Adapter 就不再是"一组数"，而是一个有确定几何位置的"视角"。不同 Adapter 之间的关系——相似还是疏远——现在有了内蕴的、不依赖坐标选择的度量。这是从"统计思维"到"几何思维"的关键一步：距离不再由启发式定义，而是由流形的黎曼度量直接给出。

### Page 5 — 李群与 Stiefel 流形的黎曼几何

**合并思路**：原 Page 7 独立成页。这是全文公式最集中、数学最深的一页——建立李群→齐性空间→Stiefel流形→测地线距离的完整推导链。

**内容要点**：
- 李群 G 的定义：既是群又是光滑流形，群运算光滑
- 对 ML 的意义：(a) 自然黎曼度量 (b) 闭合测地线方程 (c) 群作用描述合法参数变换
- Stiefel 流形的群论结构：St(d,r) ≅ SO(d) / SO(d-r)
  - SO(d) 通过左乘传递地作用于 St(d,r)
  - 稳定子群 SO(d-r) 作用在未使用的 d-r 维上
  - 商空间继承的黎曼度量 = 我们使用的测地线距离
- 核心公式 — 测地线距离：
  $$d_{geo}(W_1, W_2) = \|\arccos(\sigma(W_1^T W_2))\|_2$$
  - σ 是 W₁^T W₂ 的奇异值，arccos 按元素作用
  - σᵢ = cos(θᵢ)，θᵢ = 两个子空间的**主角度**（principal angles）
- 几何直觉：θᵢ=0 表示子空间重合，θᵢ=π/2 表示正交。d_geo 是子空间差异的综合度量

**建议图示**：左侧=一个圆表示 SO(d)，内部用不同颜色划分成陪集，每个陪集标注一个箭头指向右侧 Stiefel 流形上的一个点。右侧=Stiefel 流形曲面上画出 W₁, W₂ 两个点，之间画测地线，旁边扇形标注主角度 θ₁,...,θᵣ。

**建议动画**：SO(d) 圆先出现 → 陪集划分动画 → Stiefel 曲面从右侧滑入 → W₁,W₂ 出现 → 主角度扇形逐个画出 → 测地线公式最后以大字呈现。

**讲解词核心要点**（约 450 字）：这一页公式密集但关键——测地线距离公式是整个 Lie-SEMA 的核心。St(d,r) ≅ SO(d)/SO(d-r) 的齐性空间表示告诉我们：Stiefel 流形的几何不是随意定义的，而是由 SO(d) 的群结构（具体地，Killing 形式）唯一确定的。这意味着后续所有基于测地线的判断——扩展检测、任务差异度量——都是内蕴的，不依赖任何统计假设。

### Page 6 — 纤维丛：统一任务空间与参数空间的数学语言

**合并思路**：原 Page 8 独立成页。这是整场报告最难的概念——纤维丛的五元组定义及其到持续学习的映射。需要分层图示和精确的数学对应。

**内容要点**：
- 纤维丛的形式化定义：E = (E, B, π, F, G)
  - B（底流形）= 任务分布空间
  - F（纤维）= 在特定任务上的最优 Adapter 参数空间 ≅ St(768,16)
  - E（全空间）= 所有可能任务 × 所有可能参数
  - G（结构群）= SO(768) 的子群，描述不同任务参数间的合法变换
  - π: E → B = 将参数映射到其最适配的任务分布
- 局部平凡化：对 B 的每个局部邻域 U，π^{-1}(U) ≅ U × F
  - 含义：在相似分布上，参数空间具有相同结构 → "相似任务共享 Adapter"的几何表述
- 截面（Section）s: B → E：π ∘ s = id_B
  - 含义：一个学习算法 = 在底流形上构造截面 → 为每个任务分布选一组参数
- 联络（Connection）：定义相邻纤维之间的"平行"关系

**建议图示**：三维分层示意图——底部是一条弯曲的曲线（B = 任务分布空间），从底部每个点向上"生长出"垂直的圈/球（F = Stiefel 流形），全空间 E 是所有纤维的总和。在 E 中画一条曲线标注截面 s。用虚线框标注一个局部区域 U，放大显示 U×F 的乘积结构。用颜色区分 B（蓝灰色）、F（橙红色）、s（绿色曲线）。

**建议动画**：底流形 B 先画出 → 纤维 F 从 B 上的点逐个向上生长 → 全空间 E 形成 → 截面 s 曲线在全空间中画出 → 局部平凡化区域放大展示。

**讲解词核心要点**（约 450 字）：纤维丛为我们提供了一套语言来精确描述持续学习的本质——什么在变（底流形上的点），什么保持不变（纤维的拓扑结构），什么时候需要"跳"到新的局部平凡化邻域（触发扩展）。这不是比喻，是严格的数学对应。局部平凡化的定义 π^{-1}(U) ≅ U × F 直接告诉我们：在相似的任务分布上，可以安全复用同一个 Adapter。

### Page 7 — 纤维丛的持续学习动力学 + SEMA/Lie-SEMA 对比

**合并思路**：原 Page 9（纤维丛动力学）、原 Page 10（SEMA实现）、原 Page 11（Lie-SEMA实现）的核心内容合并。先讲纤维丛动力学映射，再讲两个系统的具体实现，最后做两种检测范式的对比。

**内容要点**：
- 纤维丛动力学映射：
  - 顺序学习 T 个任务 = 底流形上的离散路径 p₁ → p₂ → ... → p_T
  - 在每个 pₜ 构造 s(pₜ)（找到适配的 Adapter 参数）
  - 核心决策：s(pₜ) 在已有截面"附近"（复用旧Adapter）还是需要"离开"（触发扩展）？
  - 联络定义平行移动 — d_geo < 阈值 → 共享 Adapter；d_geo > 阈值 → 新截面分支
- SEMA 实现（CVPR 2025）：
  - Functional Adapter：瓶颈 MLP（768→16→768）
  - Representation Descriptor（RD）：AE（768→128→768），Z-score 检测
  - Soft Router：Linear(768→M)→softmax → 加权 Σ α_i · Adapter_i(x)
  - 训练：func (5 epoch, CE) → rd (20 epoch, MSE) → 冻结
- Lie-SEMA 实现：
  - 核心约束：down_proj 权重 W ∈ St(768,16)，W^T W = I₁₆
  - 初始化：Haar 分布采样（QR 分解随机高斯矩阵）
  - 优化：Riemannian SGD — 每步 SGD 后 SVD 投影 W ← U V^T（~200K FLOPs）
  - 扩展检测：$d_{geo}(W_{new}, W_i) = \|\arccos(\sigma(W_{new}^T W_i))\|_2$
  - 不再需要 RD（AE）训练 → 节省 20 epochs 训练开销

**建议图示**：上半部分=纤维丛三维图（同 Page 6），叠加离散路径 p₁→p₂→p₃ 动画。下半部分=两列对比（SEMA vs Lie-SEMA），左列标注数据流（输入→RD→Z-score→判定），右列标注数据流（输入→训练Adapter→SVD投影→测地线→判定），用橙/绿/蓝标注三组件对应关系，用粗箭头=数据流，虚线=控制流。

**建议动画**：纤维丛路径动画先展示 → 页面下滑 → SEMA 和 Lie-SEMA 两列同时展开 → 数据流箭头逐段画出 → 控制流虚线最后标注。

**讲解词核心要点**（约 480 字）：这一页完成了从数学语言到机器学习操作的全映射。纤维丛的三个核心操作——平行移动（联络）、距离测量（测地线）、截面分支（扩展）——分别对应微调、扩展检测、新增Adapter。SEMA 的统计检测链（数据→AE→Z-score）与 Lie-SEMA 的几何检测链（数据→SVD→d_geo）构成"统计间接 vs 几何直接"的对比。Lie-SEMA 不再需要 RD 训练阶段（20 epochs × 197K 参数），这是几何方法带来的工程收益。

### Page 8 — 实验结果与纤维丛理论启示

**内容要点**：

  1. 底流形上的距离 = 任务差异的严格数学定义
