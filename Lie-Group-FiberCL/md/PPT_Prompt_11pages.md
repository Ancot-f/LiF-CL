## 二、背景资料

### 资料 A：机器学习理论基础（来源：李凡长《机器学习理论及应用》）

本书建立了机器学习理论的完整框架，你需要从中提取以下关键数学概念作为 PPT 前 1/3 的理论基础：

**概念1：PAC 学习理论（Probably Approximately Correct）**

- 核心思想：一个概念类是 PAC 可学习的，当且仅当存在一个算法，对任意 ε, δ > 0，当训练样本数 m 足够大时，学习器能以至少 1-δ 的概率输出一个泛化误差不超过 ε 的假设
- 关键公式：样本复杂度下界 $m \geq \frac{1}{\epsilon}(\ln|H| + \ln\frac{1}{\delta})$
- 需要强调的洞察：PAC 理论保证泛化的前提是——训练集和测试集来自**同一个分布 D**。这是所有后续讨论的逻辑起点。

**概念2：VC 维（Vapnik-Chervonenkis Dimension）**

- 核心思想：度量假设空间的"表达能力"——一个假设空间 H 的 VC 维是能被 H 打散的最大样本数
- 关键公式：$m \geq \frac{1}{\epsilon}(d(\ln\frac{2m}{d}+1) + \ln\frac{4}{\delta})$，其中 d 是 VC 维
- 需要强调的洞察：VC 维越大 → 模型表达能力越强 → 但泛化所需的样本也越多。这为后续讨论"模型按需增容"提供了理论伏笔。

**概念3：结构风险最小化（Structural Risk Minimization, SRM）**

- 核心思想：将假设空间按 VC 维组织成嵌套结构 S₁ ⊂ S₂ ⊂ ... ⊂ Sₖ，在经验风险和置信范围之间寻找最优平衡
- 关键公式：$R(f) \leq R_{emp}(f) + \Phi(\frac{h}{n})$
- 需要强调的洞察：当数据量固定时，不能无限制增大模型容量。在持续学习中，"何时增容"成为需要严格理论依据的决策。

**概念4：独立同分布假设（i.i.d. Assumption）的统治地位**

- PAC、VC维、SRM 的全部理论保证都建立在训练数据和测试数据独立同分布这一假设之上
- 这是经典统计学习理论的"阿基米德支点"——抽掉它，整个框架需要被重新审视
- 也是我们要系统性地挑战和扩展的假设

### 资料 B：李群与纤维丛的数学基础

以下是从李凡长《机器学习理论及应用》中李群纤维丛学习部分提取的核心数学结构。你需要将这些内容融入 PPT 的理论部分。

**概念5：李群（Lie Group）基础**

- 定义：李群 G 是一个既是群又是光滑流形的数学对象，群运算（乘法和取逆）是光滑映射
- 对机器学习的关键意义：李群同时具有代数结构（群运算 = 参数组合）和几何结构（流形 = 参数空间的局部欧氏性质），这使它成为描述"结构化参数空间"的自然语言
- 矩阵李群的核心例子：
  - 一般线性群 GL(n)：所有 n×n 可逆矩阵
  - 特殊正交群 SO(n)：{R ∈ ℝ^{n×n} | R^T R = I, det(R) = 1}
  - Stiefel 流形 St(d,r)：{W ∈ ℝ^{d×r} | W^T W = I_r}
- Stiefel 流形的齐性空间表示：St(d,r) ≅ SO(d) / SO(d-r)，即它是 SO(d) 的一个商空间。这个表示意味着 Stiefel 流形继承了一个自然的黎曼度量——由 SO(d) 的 Killing 形式诱导

**概念6：Stiefel 流形的黎曼几何**

- 切空间：在 W ∈ St(d,r) 处的切空间为 T_W St(d,r) = {ξ ∈ ℝ^{d×r} | ξ^T W + W^T ξ = 0}，即所有使正交约束一阶保持的扰动方向
- 黎曼度量：⟨ξ, η⟩_W = tr(ξ^T (I - ½W W^T) η)，它是 SO(d) 上 Killing 形式在商空间上的诱导度量
- 测地线方程：Stiefel 流形上存在闭合形式的测地线（由 Edelman et al., 1998 给出），这允许我们精确计算流形上两点之间的最短路径
- 测地线距离公式：
  - 设 W₁, W₂ ∈ St(d,r)，计算 W₁^T W₂ ∈ ℝ^{r×r} 的奇异值 σ₁, ..., σᵣ
  - 主角度（principal angles）：θᵢ = arccos(σᵢ)，其中 σᵢ = cos(θᵢ)
  - 测地线距离：$d_{geo}(W_1, W_2) = \sqrt{\sum_{i=1}^{r} \theta_i^2}$ （主角度向量的 ℓ² 范数）
- 几何直觉：两个 Stiefel 点之间的测地线距离度量了它们所张成的 r 维子空间之间的"夹角"。θᵢ = 0 表示两个子空间重合；θᵢ = π/2 表示正交。测地线距离是这些"子空间差异"的综合度量

**概念7：纤维丛（Fiber Bundle）理论框架**

- 定义：纤维丛 E 是一个五元组 (E, B, π, F, G)，其中：
  - E：全空间（Total Space）— 所有的模型参数
  - B：底流形（Base Space / Base Manifold）— 任务分布空间，每个点代表一种数据分布
  - π: E → B：投影映射 — 将参数映射到其适用的任务
  - F：标准纤维（Typical Fiber）— 在单个任务上的模型参数空间
  - G：结构群（Structure Group）— 一个李群，作用于 F 上，描述不同纤维之间的变换关系
- 局部平凡化（Local Trivialization）：对 B 的每个局部邻域 U，π^{-1}(U) ≅ U × F。即局部来看，纤维丛就是笛卡尔积——这意味着在相似的分布上，参数空间具有相同的结构
- 截面（Section）：映射 s: B → E，满足 π ∘ s = id_B。在机器学习中，一个"学习算法"就是在底流形（任务分布空间）上构造一个截面——为每个任务分配一组参数
- 联络（Connection）：定义了相邻纤维之间的"平行"关系，允许我们在底流形上沿路径"平行移动"纤维中的元素

**概念8：纤维丛视角下的持续学习**

- 底流形 B = 任务分布空间：每个任务对应底流形上的一个点。相似的任务在底流形上彼此靠近，差异大的任务远离
- 纤维 F_p = 在任务 p 上的最优参数：对于每个任务分布 p ∈ B，存在一组最优的 Adapter 参数。这组参数位于 p 上方的纤维中
- 结构群 G = SO(d)（或其在 Stiefel 流形上的作用）：描述不同任务参数之间的合法变换。G 的结构决定了参数空间的几何——测地线距离、平行移动、曲率
- 持续学习的纤维丛表述：
  - 顺序学习任务 T₁, T₂, ..., Tₜ 等价于在底流形上走一条分段路径 p₁ → p₂ → ... → pₜ
  - 在每个新任务 pₜ₊₁ 处，需要构造一个截面 s(pₜ₊₁)（即找到该任务的参数）
  - 核心问题：s(pₜ₊₁) 应该在多大程度上复用已有截面的信息（即旧 Adapter），还是应该在纤维中"离开"已有的截面（即触发扩展）？
  - 测地线距离回答了这个问题：如果新参数与所有旧参数在纤维上的测地线距离都很大（超过阈值），说明新任务在底流形上距离所有旧任务都远——需要新的截面分支（新 Adapter）
- 与经典理论的统一：
  - PAC 的 i.i.d. 假设：对应到底流形上，就是假设所有数据来自同一个点 p。在这种退化情况下，纤维丛退化为一个平凡的笛卡尔积，不需要联络，不需要平行移动
  - SEMA 的分布偏移检测（Z-score）：是用统计方法估计底流形上的两个点是否"足够近"
  - Lie-SEMA 的测地线距离检测：直接用纤维丛的内蕴几何来判断——不需要统计估计，距离由结构群 G 的黎曼度量直接给出

### 资料 C：SEMA 与 Lie-SEMA 的具体实现

**SEMA（Self-Expansion with Mixture of Adapters）— CVPR 2025**

三个核心组件：

1. **Functional Adapter**：瓶颈 MLP（768 → 16 → 768），插入 ViT 第 9-11 层的 MLP 旁
2. **Representation Descriptor（RD）**：轻量自编码器（768 → 128 → 768），用 Z-score 检测分布偏移
3. **Soft Router**：Linear(768 → M) → softmax，动态加权组合所有 Adapter 的输出

训练流程：检测（判断是否扩展）→ func 阶段（5 epoch, CE loss）→ rd 阶段（20 epoch, MSE loss）→ 冻结

**Lie-SEMA（李群改进版）**

核心改进：将 Adapter 的 down_proj 权重 W ∈ ℝ^{768×16} 约束到 Stiefel 流形 St(768,16) = {W | W^T W = I₁₆}

三个关键变化：

1. **参数初始化**：从 Haar 分布采样随机正交矩阵（QR 分解），而非 Kaiming Uniform
2. **优化约束**：每次 SGD 步后执行 SVD 投影 W ← U V^T，将 W 拉回 Stiefel 流形表面。这等价于 Riemannian SGD
3. **扩展检测升级**：从 Z-score（统计检测）→ 测地线距离（几何检测）
   - 计算公式：新 Adapter 训练完成后，计算 `min_i d_geo(W_new, W_i)`，若 > geo_threshold → 触发扩展
   - 本质上是在纤维中测量新旧"截面"的内蕴距离，而非依赖 AE 的间接统计

- 标准深度学习的参数空间 = ℝ^d（欧氏空间），优化 = 沿负梯度移动
- 问题：欧氏距离 ‖W₁ - W₂‖_F 可能很小，但描述的特征子空间完全不同
- 引入约束：要求 W^T W = I_r → 参数不再生活在 ℝ^{d×r} 中
- Stiefel 流形定义：St(d,r) = {W ∈ ℝ^{d×r} | W^T W = I_r}
- 几何直觉：W 的 r 个列向量是 ℝ^d 中的正交归一基。W 指定了一个 r 维子空间（附带基底选择）。不同 Adapter 学习不同的 r 维"视角"来投影特征
- 测地线 vs 直线：欧氏距离的直线穿过流形内部（无意义），测地线沿流形表面（合法路径）

**建议图示**：左侧=欧氏空间中两个点之间的直线（红色虚线穿过流形内部，标注"‖W₁-W₂‖_F = 小，但子空间差异大"）。右侧=Stiefel 流形（弯月形曲面），上面两个点，测地线沿曲面连接（实线），标注"d_geo(W₁,W₂) = 大"。两个 W 用小方阵图标表示，标注维度 [768×16]。

**建议动画**：先画出欧氏空间和直线 → 直线变红虚线（表示无效） → Stiefel 曲面从左下方"生长"出来 → 两个点出现在曲面上 → 测地线沿曲面逐步描出。

**讲解词核心要点**（约 350 字）：当我们约束 W 到 Stiefel 流形，一个 Adapter 就不再是"一组数"，而是一个有确定几何位置的"视角"。不同 Adapter 之间的关系——相似还是疏远——现在有了内蕴的、不依赖坐标选择的度量。这是从"统计思维"到"几何思维"的关键一步：距离不再由启发式定义，而是由流形的黎曼度量直接给出。

### Page 5 — 李群与 Stiefel 流形的黎曼几何

**设计意图**：全文公式最集中、数学最深的一页——建立李群→齐性空间→Stiefel流形→测地线距离的完整推导链。

**内容要点**：
- 李群 G 的定义：既是群又是光滑流形，群运算光滑
- 对 ML 的意义：(a) 自然黎曼度量 (b) 闭合测地线方程 (c) 群作用描述合法参数变换
- Stiefel 流形的群论结构：St(d,r) ≅ SO(d) / SO(d-r)
  - SO(d) 通过左乘传递地作用于 St(d,r)
  - 稳定子群 SO(d-r) 作用在未使用的 d-r 维上
  - 商空间继承的黎曼度量 = 我们使用的测地线距离
- 核心公式 — 测地线距离：
  $$d_{geo}(W_1, W_2) = \|\arccos(\sigma(W_1^T W_2))\|_2$$
  - σ 是 W₁^T W₂ 的奇异值，arccos 按元素作用
  - σᵢ = cos(θᵢ)，θᵢ = 两个子空间的**主角度**（principal angles）
- 几何直觉：θᵢ=0 表示子空间重合，θᵢ=π/2 表示正交。d_geo 是子空间差异的综合度量

**建议图示**：左侧=一个圆表示 SO(d)，内部用不同颜色划分成陪集，每个陪集标注一个箭头指向右侧 Stiefel 流形上的一个点。右侧=Stiefel 流形曲面上画出 W₁, W₂ 两个点，之间画测地线，旁边扇形标注主角度 θ₁,...,θᵣ。

**建议动画**：SO(d) 圆先出现 → 陪集划分动画 → Stiefel 曲面从右侧滑入 → W₁,W₂ 出现 → 主角度扇形逐个画出 → 测地线公式最后以大字呈现。

**讲解词核心要点**（约 380 字）：这一页公式密集但关键——测地线距离公式是整个 Lie-SEMA 的核心。St(d,r) ≅ SO(d)/SO(d-r) 的齐性空间表示告诉我们：Stiefel 流形的几何不是随意定义的，而是由 SO(d) 的群结构（具体地，Killing 形式）唯一确定的。这意味着后续所有基于测地线的判断——扩展检测、任务差异度量——都是内蕴的，不依赖任何统计假设。

### Page 6 — 纤维丛：统一任务空间与参数空间的数学语言

**设计意图**：整场报告最难的概念——纤维丛的五元组定义及其到持续学习的映射。需要分层图示和精确的数学对应。

**内容要点**：
- 纤维丛的形式化定义：E = (E, B, π, F, G)
  - B（底流形）= 任务分布空间
  - F（纤维）= 在特定任务上的最优 Adapter 参数空间 ≅ St(768,16)
  - E（全空间）= 所有可能任务 × 所有可能参数
  - G（结构群）= SO(768) 的子群，描述不同任务参数间的合法变换
  - π: E → B = 将参数映射到其最适配的任务分布
- 局部平凡化：对 B 的每个局部邻域 U，π^{-1}(U) ≅ U × F
  - 含义：在相似分布上，参数空间具有相同结构 → "相似任务共享 Adapter"的几何表述
- 截面（Section）s: B → E：π ∘ s = id_B
  - 含义：一个学习算法 = 在底流形上构造截面 → 为每个任务分布选一组参数
- 联络（Connection）：定义相邻纤维之间的"平行"关系

**建议图示**：三维分层示意图——底部是一条弯曲的曲线（B = 任务分布空间），从底部每个点向上"生长出"垂直的圈/球（F = Stiefel 流形），全空间 E 是所有纤维的总和。在 E 中画一条曲线标注截面 s。用虚线框标注一个局部区域 U，放大显示 U×F 的乘积结构。用颜色区分 B（蓝灰色）、F（橙红色）、s（绿色曲线）。

**建议动画**：底流形 B 先画出 → 纤维 F 从 B 上的点逐个向上生长 → 全空间 E 形成 → 截面 s 曲线在全空间中画出 → 局部平凡化区域放大展示。

**讲解词核心要点**（约 380 字）：纤维丛为我们提供了一套语言来精确描述持续学习的本质——什么在变（底流形上的点），什么保持不变（纤维的拓扑结构），什么时候需要"跳"到新的局部平凡化邻域（触发扩展）。这不是比喻，是严格的数学对应。局部平凡化的定义 π^{-1}(U) ≅ U × F 直接告诉我们：在相似的任务分布上，可以安全复用同一个 Adapter。

### Page 7 — SEMA 架构详解：三组件的协同设计（CVPR 2025）

**设计意图**：这是从数学理论回到工程实现的关键页。在引入纤维丛语言之后，展示 SEMA 作为"不自觉的纤维丛实现"——它的三组件设计恰好对应了纤维丛的核心操作，只是用的是统计工具而非几何工具。详细的架构图、维度标注和三组件交互流程为下一页的"映射论证"提供具体的工程锚点。

**内容要点**：
- 整体架构：冻结 ViT-B/16 主干，在 12 层 Transformer 的 MLP 旁插入 SEMA Module
  - 插入位置：第 9-11 层（深层特征，语义抽象度高）
  - 每层各有独立的 SEMA Module，层间不共享参数
- 组件一：Functional Adapter（橙色）
  - 结构：瓶颈 MLP — Linear(768→16) + GELU + Linear(16→768)
  - down_proj W ∈ ℝ^{768×16}（Kaiming Uniform 初始化），up_proj 全零初始化
  - 残差连接：Adapter(x) = up_proj(GELU(down_proj(x))) + x
  - 角色：存储任务特定的知识，每个 Adapter 对应一个任务的参数化"视角"
- 组件二：Representation Descriptor / RD（绿色）
  - 结构：轻量自编码器 — Linear(768→128) + Linear(128→768)
  - 训练方式：MSE 重构损失，仅在新任务数据上训练
  - Z-score 计算：对每个旧 RD，计算当前数据的重建误差，若 |error - μ_old|/σ_old > 2.0 → 判定分布偏移
  - 角色：分布偏移的统计检测器——隐式地估计"底流形上的距离"
- 组件三：Soft Router（蓝色）
  - 结构：Linear(768 → M) → softmax，其中 M = 当前 Adapter 总数
  - 输出：加权组合 Σ_{i=1}^{M} α_i · Adapter_i(x)，α_i 由 Router 根据输入特征动态分配
  - 角色：跨任务的知识复用——在多个"纤维截面"之间做软插值
- 训练流程：
  - 检测阶段：用所有旧 RD 评估 Z-score → 判断是否全部超过阈值
  - func 阶段（5 epoch, CE loss）：训练新 Adapter + Router
  - rd 阶段（20 epoch, MSE loss）：训练新 RD，建立当前分布的统计基准
  - 冻结：完成后所有新参数冻结，加入"已学任务集合"

**建议图示**：整页分为三栏。左栏=ViT 整体结构（12层堆叠），在第9-11层处用橙色高亮标注 SEMA 插入点。中栏=单层 SEMA Module 的放大图——输入特征 x [B,768] 分三路进入三个组件：(1) 橙色 Adapter 路径：x→down_proj[768→16]→GELU→up_proj[16→768]→残差加回；(2) 绿色 RD 路径：x→encoder[768→128]→decoder[128→768]→重建误差；(3) 蓝色 Router 路径：x→Linear[768→M]→softmax→加权 Σ α_i Adapter_i(x)。所有数据流用粗箭头（颜色与模块匹配），关键位置标注维度。右栏=训练流程状态机：检测→func(5ep)→rd(20ep)→冻结。

**建议动画**：ViT 整体结构先出现 → 第9-11层高亮 → 单层 SEMA Module 放大弹出 → 三个组件逐个出现（Adapter→RD→Router）→ 数据流箭头逐段画出 → 右栏训练状态机从检测开始逐步推进。

**讲解词核心要点**（约 380 字）：SEMA 三组件各司其职——Adapter 存储"视角"，RD 感知"分布变化"，Router 实现"知识复用"。注意两个关键设计：Adapter 使用瓶颈结构（768→16→768），16 维瓶颈并非随意选择——后续将看到，16 维子空间在 Stiefel 流形上的测地线距离恰好能有效区分不同任务。RD 用 AE 的重建误差作为分布偏移的代理信号——这是一种统计近似，它"在做什么"是对的（测量任务间差异），但"怎么做"依赖正态假设和 AE 训练质量。这一点将成为从 SEMA 到纤维丛映射的切入点。

### Page 8 — 从 SEMA 到纤维丛：一个必然的数学映射

**设计意图**：**全文最重要的桥梁页**。逐一论证 SEMA 的三个组件和两个核心操作恰好是纤维丛五要素 (E, B, π, F, G) 和联络/截面操作的隐式实现。核心论点是：SEMA 之所以有效，是因为它的工程设计在无意中逼近了纤维丛的结构——但用的是统计近似工具。由此自然引出"如果将统计代理替换为几何真值会怎样？"——这就是 Lie-SEMA 的动机。

**内容要点**：
- SEMA 组件 → 纤维丛要素的精确映射：
  - **B（底流形）= RD 检测的分布偏移空间**：每个 RD 的 Z-score 基准 (μ, σ) 定义了底流形上的一个点。当 Z-score > 阈值，意味着新数据在底流形上远离所有已知点 → 需要探索新的纤维
  - **F（纤维）= Adapter 参数空间**：每个 Adapter 的 down_proj W ∈ ℝ^{768×16}，位于 St(768,16) 上。不同 Adapter 位于不同任务点上方的纤维中
  - **π: E → B（投影）= RD 的分布识别**：将一组 Adapter 参数映射到它最适配的任务分布——由 RD 的重建误差决定
  - **G（结构群）= 参数空间的合法变换**：SEMA 中未显式约束 → Adapter 权重自由在 ℝ^{768×16} 中移动 → 无法保证内蕴距离的有效性
  - **截面 s = 学习算法**：SEMA 的训练流程（检测→func→rd→冻结）就是在底流形上逐点构造截面
- SEMA 操作 → 纤维丛操作的映射：
  - **局部平凡化 = 共享 Adapter**：Z-score < 阈值 → 新旧任务落在同一局部邻域 U → π^{-1}(U) ≅ U × F → 可复用已有 Adapter
  - **扩展 = 新截面分支**：Z-score > 阈值（对所有旧 RD）→ 新任务不在任何已有局部邻域内 → 必须创建新 Adapter（截面分叉）
  - **Router 加权 = 纤维间的插值**：Router 的 softmax 权重 α_i 本质上是在不同纤维截面之间做凸组合——一种"软"的知识迁移
- 关键洞察 — SEMA 的"无意识的纤维丛"：
  - SEMA 设计者可能没有显式使用纤维丛语言，但三组件结构恰好复现了纤维丛的核心操作
  - 这解释了 SEMA 的有效性：它的工程设计符合了底流形-纤维-截面的几何结构
  - 也揭示了 SEMA 的局限：RD（AE + Z-score）是统计代理 → 依赖正态假设、需要训练 AE（20 epochs）、对缓冲区大小敏感
- 自然的问题：如果去掉统计代理，直接用纤维丛的内蕴几何会怎样？
  - → 将 Adapter 权重约束到 Stiefel 流形 → 结构群 G 显式化
  - → 用测地线距离替代 Z-score → 距离由黎曼度量直接给出
  - → 这就是 Lie-SEMA

**建议图示**：整页采用"上映射 + 下对比"布局。上半部分=左侧列出 SEMA 的三个组件（Adapter/RD/Router，用对应颜色），中间用"="号和箭头连接，右侧列出纤维丛五要素 (B, F, π, G, 截面)，每个对应关系用相同颜色的连线标注。下半部分=两列对比——左列"SEMA（统计代理）"→ 数据→AE→Z-score→阈值（绿色框），右列"Lie-SEMA（几何真值）"→ 数据→训练→SVD→d_geo→阈值（橙色框），中间标注大问号"？"作为过渡到下一页的引子。

**建议动画**：上半部分逐对出现映射关系（B↔RD, F↔Adapter, π↔分布识别, G↔?→显式化, 截面↔训练流程）→ G 的映射用闪烁问号标注"未约束"→ 下半部分两列同时展开 → 中间问号放大 → 过渡句："如果把这个统计代理换成几何真值呢？"

**讲解词核心要点**（约 400 字）：这一页是全文的逻辑枢纽。SEMA 不是一个随意设计的工程系统——它的三组件结构恰好实现了纤维丛的核心操作，只是使用了统计工具作为代理。RD 的 Z-score 是在用统计方法估计底流形上的距离，Adapter 的瓶颈结构（768→16）天然定义了一个 Grassmann/Stiefel 空间，Router 的 softmax 加权就是在不同纤维截面之间做插值。SEMA 的"无意识的纤维丛结构"解释了它为什么有效，也暴露了它的理论脆弱点：Z-score 依赖正态假设和 AE 训练质量，结构群 G 没有显式约束（Adapter 权重在欧氏空间中自由移动），导致距离度量不内蕴。这自然引出下一步——如果我们显式地引入李群和 Stiefel 流形约束，将统计代理替换为几何真值，会发生什么？

### Page 9 — 纤维丛动力学 + Lie-SEMA：从统计代理到几何真值

**设计意图**：将纤维丛的动力学描述与 Lie-SEMA 的几何升级合并。先讲纤维丛的三个核心操作如何映射到持续学习流程，再展示 Lie-SEMA 如何通过三个关键改动（Haar初始化、Riemannian SGD、测地线检测）将统计代理替换为几何真值。

**内容要点**：
- 纤维丛动力学映射：
  - 顺序学习 T 个任务 = 底流形上的离散路径 p₁ → p₂ → ... → p_T
  - 在每个 pₜ 构造 s(pₜ)（找到适配的 Adapter 参数）
  - 核心决策：s(pₜ) 在已有截面"附近"（复用旧Adapter）还是需要"离开"（触发扩展）？
  - 联络定义平行移动 — d_geo < 阈值 → 共享 Adapter；d_geo > 阈值 → 新截面分支

  1. **参数初始化**：从 Haar 分布采样（QR 分解随机高斯矩阵）→ 初始 W 严格位于 Stiefel 流形上
  2. **优化约束**：Riemannian SGD — 每步 SGD 后 SVD 投影 W ← U V^T（~200K FLOPs）→ W 始终在 Stiefel 上
  3. **扩展检测升级**：$d_{geo}(W_{new}, W_i) = \|\arccos(\sigma(W_{new}^T W_i))\|_2$

     - **不再需要 RD（AE）训练** → 节省 20 epochs × 197K 参数的训练开销
- 对比总结：
  - SEMA：数据→AE→重建误差→Z-score→阈值判定（统计间接）
  - Lie-SEMA：数据→训练临时Adapter→SVD→测地线距离→几何阈值判定（几何直接）
  - 结构群 G：SEMA 中未显式定义 → Lie-SEMA 中由 SO(768)/SO(752) 显式约束

### Page 10 — 实验结果与纤维丛理论启示

**内容要点**：
- 理论回顾 — 经典理论的纤维丛扩展：
  - PAC → 当所有数据来自同一点 p∈B 时有效；分布漂移时需通过联络平行移动
  - VC 维 → 决定单个截面在底流形上的"覆盖范围"；超出 → 需新截面分支（新 Adapter）
  - SRM → 截面分支的选择原则：d_geo 过大时，保持 h 导致 R_emp↑，增加 h 导致 Φ↑，最优 = 平衡
  - 向后兼容：当底流形退化为单点 → 框架退化为标准 PAC/SRM
- 三点核心贡献：
  1. **从统计到几何**：不需要分布假设，距离由流形内蕴几何决定（测地线距离替代 Z-score）
  2. **从人工预设到数据驱动**：扩展由底流形上的测地线距离自动判断（不再"每N个任务加一个模块"）
  3. **从工程方案到理论框架**：B/F/G/联络 → 任务空间/参数空间/合法变换/知识迁移，形成自洽的数学体系
  - SEMA 是"不自觉的纤维丛" → Lie-SEMA 是"自觉的纤维丛实现"
- 四个开放问题：
  1. 底流形 B 的拓扑能否从数据中学习？（而非预设为欧氏空间）
  2. 联络能否从数据中学习？（超越标准 Levi-Civita 联络，捕获任务间的非平凡依赖）
  3. 纤维丛的曲率是否对应"遗忘的不可逆性"？（曲率 ≠ 0 → 平行移动路径依赖 → 遗忘不可逆）
  4. 能否推广到更一般的结构群（SE(n), Sp(n)）？（处理不同约束下的参数空间）
